# Что за библиотека dbschemix

Это инструмент для безопасного управления изменениями схемы и содержимого баз данных.

## Фреймворк агностик

Можно использовать в любых проектах, в том числе если у вас Laravel или Yii, 
но вам не нравится Active Record или Query Builder в миграциях.

## Чистый SQL

Для написания миграций используется чистый SQL.

## Кастомизация драйвера

По умолчанию использует PDO драйвер для работы с СУБД, но можно заменить на что-то своё 

## Версионирование

Миграции выполненные в рамках одной интерпретации `up`, будут иметь признак версии, 
что даёт возможность откатить `down` их одной командой, не анализируя историю
